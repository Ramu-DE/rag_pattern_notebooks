## 🎯 Overview

This repository contains **37 production-ready RAG (Retrieval-Augmented Generation) patterns** implemented with AWS Bedrock, Amazon Titan, OpenSearch Serverless, and DynamoDB.

### What is RAG?

RAG combines Large Language Models (LLMs) with external knowledge retrieval to generate accurate, context-aware responses.
